Comprendre l’intelligence artificielle en la voyant fonctionner

SUNLAB IA propose des ateliers interactifs permettant de découvrir les réseaux de neurones, les embeddings, les Transformers, les agents IA et les grands concepts de l’intelligence artificielle moderne.

Objectif du projet

Ces ateliers ont été conçus pour rendre les concepts d’intelligence artificielle accessibles au grand public, aux FabLabs, aux enseignants, aux étudiants et aux jeunes.

Chaque atelier combine :

  • des démonstrations visuelles
  • du code Python interactif
  • des explications progressives
  • une approche pédagogique concrète
Les ateliers
1

Réseaux de neurones

Découvrir comment un réseau de neurones apprend progressivement à transformer des entrées simples en prédictions probabilistes.

Voir l’atelier Code Python

Vecteurs & embeddings

Voir l’atelier Code Python

Attention & Transformers

Explorer les mécanismes d’attention qui permettent aux modèles modernes comme ChatGPT de comprendre les relations entre les mots.

Voir l’atelier Code Python
